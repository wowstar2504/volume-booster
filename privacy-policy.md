# Volume Booster – Privacy Policy

**Effective Date:** February 28, 2026

Volume Booster respects your privacy. This extension does not collect, store, transmit, or sell any personal data.

---

## 1. Data Collection

Volume Booster does **not** collect:

- Personal information
- Browsing history
- Account information
- Analytics data
- Tracking data
- Usage statistics

All functionality operates locally within the user's browser.

---

## 2. Permissions Usage

The extension uses the following Chrome permissions:

### activeTab
Used to apply volume amplification to the currently active tab when the user interacts with the extension.

### scripting
Used to inject scripts that modify audio elements on webpages to enable volume boosting.

### storage
Used to locally store the user’s selected volume level so it persists between page reloads and browser sessions.  
This data is stored only in the browser and is never transmitted externally.

### declarativeNetRequest & Host Permission (https://flavortown.hackclub.com/*)
Used to attach a required HTTP request header:

X-Flavortown-Ext-{project_id}: true

This header is required for project registration with Hack Club Flavortown.  
No browsing data or personal information is accessed, read, or transmitted.

---

## 3. Data Sharing

Volume Booster does not share any user data with third parties.  
No data is sold, transferred, or disclosed.

---

## 4. Remote Code

This extension does not use remote code.  
All scripts and logic are packaged within the extension and executed locally in the user's browser.
